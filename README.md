# ImageArea

### 相关框架
  - 前端：HTML5 + CSS3 + Jade(Pug)
  - 后端：Node.js
  - 前端框架：Bootstrap4 
  - Web框架：express框架 
  - 本地数据库：MongoDB(存储⽤用户基本信息)
  - 云端数据库：OSS(存储⽤用户的图⽚片)


### 功能（不完全列举）
  - 前后端的图像管理
  - 用户管理
    - 登录注册相关
    - 个人信息相关
  - 图像管理
    - 图像的上传（单张与多张）
    - 图像标签、权限、水印、相册种类
    - 评论点赞他人图像，查看他人图像
    - ......   
  - 简单搜索
 
  
### 创意与设计
  - 标签云
  - 基于深度学习的图像识别并⾃自动添加标签（MoblieNet模型）
  - 较好的交互（拖动删除、点赞动画）
